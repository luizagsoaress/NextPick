Sobre o site
Esta página web foi desenvolvida como um projeto de estudo sobre HTML5, CSS e JavaScript. O banco de dados utilizado para a autenticação dos usuários e para o armazenamento de informações foi o Firebase. O codigo com as chaves privadas foi hospedado em um servidor Node.js com Express, que está em execução no Render.
